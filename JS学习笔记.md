#### 1.Js的注意事项

### p26 字符串转成数字类型 - 方式二

- parseInt(string,radix)函数：将字符串转换成整数类型，radix表示基数，这里可以理解为进制

  - 如果第一个字符是数字或运算符号，那么就开始解析，知道遇到非数字字符，停止解析并得到解析结果
  - 如果第一个字符是非数字且非运算符号，则不解析并得到结果NaN

- parseFloat(string)函数：将字符串转换成浮点类型（小数类型）

  - 如果第一个字符是数字或运算符号，那么就开始解析，知道遇到非数字字符，停止解析并得到解析结果
  - 如果第一个字符是非数字且非运算符号，则不解析并得到结果NaN

  例如：`parseInt("abc")或parseInt("123abc")` 第一个是NaN，第二个是123

### p59 循环的跳转 -break - continue

- 循环的跳转
  - 在执行循环过程中，遇到某一个条件时，我们可能想要做一些事情
  - 比如循环体不再执行（即使没有执行完），跳出循环
  - 比如本次循环体不再执行，执行下一次的循环体
- 循环的跳转控制
  - break：直接跳出循环，循环结束
    - break某一条件满足时，退出循环，不再执行后续重复的代码
    - continue：跳出本次循环次，执行下一次循环体
      - continue某一条件满足时，不执行后续重复的代码

### p90 立即执行函数的写法

- 什么是立即执行函数？
  
- 专业名字：Immediately- Invoked function expression（IIFE 立即调用函数表达式）
  
- 就是说 一个函数定义完后被立即执行

  - 第一部分是定义了一个匿名函数，这个函数有自己独立的执行上下文环境
  - 第二部分是后面的（），表示这个函数被执行了

  ```javascript
  function(){
    console.log("立即执行函数")
  }()
  ```

- 作用：会创建一个独立的执行上下文环境，可以避免外界访问或修改内部的变量，也避免了对内部变量的修改

### p91 值传递和引用传递

```javascript
function fn(name) {
    name = "lily"
}
var temp = "eva";
fn(temp)
console.log(temp) //eva
```

当值传递 到函数里 ，最后没有改变

如果是引用类型的，比如数组，就会改变

主要原因还是堆空间和栈空间存放的地方不同

函数调用栈也是放在栈中的，运行后就释放。一开始是只有外面定义的temp，然后有加了函数调用栈，函数释放后，留在栈空间的就还是原来的那个。所以打印出来的还是temp，留下的

### 数组方法是否改变原数组 

**不改变原数组**

| 方法名     | 作用                             | 返回值                   |
| ---------- | -------------------------------- | ------------------------ |
| concat()   | 连接两个或多个数组               | 返回被连接数组的一个副本 |
| join()     | 把数组中的所有元素放入一个字符串 | 返回字符串               |
| slice()    | 从已有的数组中返回特定的元素     | 返回一个新数组           |
| toString() | 把数组转为字符串                 | 返回数组的字符串形式     |

**改变原数组**

| 方法名    | 作用                                                         | 返回值                |
| --------- | ------------------------------------------------------------ | --------------------- |
| pop()     | 删除数组最后一个元素,若数组为空,则不改变数组,返回undefined   | 返回被删除的元素      |
| push()    | 向数组末尾添加一个或多个元素                                 | 返回 **新数组的长度** |
| shift()   | 把数组的第一个元素删除,若数组为空,则不改变数组,返回undefined | 返回第一个元素的值    |
| unshift() | 向数组的开头添加一个或多个元素                               | 返回 **新数组的长度** |
| reverse() | 颠倒数组中元素的顺序                                         | 返回该数组            |
| sort()    | 对数组元素进行排序                                           | 返回该数组            |
| splice()  | 从数组中添加/删除项目                                        | 返回新数组的长度      |
